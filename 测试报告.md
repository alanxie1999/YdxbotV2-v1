## 测试内容

### 1. 简单跟随策略测试

**测试场景**：
- 6 位纯交替检测（10101/01010 反向打破）
- 正常跟随上一手结果
- 历史不足 5 手时的处理
- 无历史数据时的默认行为

**测试结果**：
```
✅ 历史=[1, 0, 1, 0, 1] → 下注=小 (6 位纯交替，反向打破)
✅ 历史=[0, 1, 0, 1, 0] → 下注=大 (6 位纯交替，反向打破)
✅ 历史=[1, 1, 0, 1, 1] → 下注=大 (跟随上一手)
✅ 历史=[0, 0, 1, 0, 0] → 下注=小 (跟随上一手)
✅ 历史=[1] → 下注=大 (历史不足 5 手，跟随)
✅ 历史=[] → 下注=大 (无历史，默认下大)
```

**结论**：6 种场景全部通过 ✅

  暂停第 2 局...

  重置：lose_count=0, bet_amount=20,000
